IPO Intelligence Dashboard — 2025

An end-to-end data analytics project analyzing 108 Indian Mainline IPOs (2025)
to help retail investors make smarter IPO investment decisions using real data

Every retail investor in India blindly applies to IPOs based on 
Grey Market Premium (GMP) tips and social media hype.
But nobody had properly analyzed:
- What % of IPOs actually give profit?
- Is it better to sell on listing day or hold long term?
- Which months give best listing gains?
- Do cheap IPOs perform better than expensive ones?



 Key Findings

 Win Rate  -- 68.52% of IPOs gave profit
 Avg Listing Gain -- 9.36% if sold on day 1
 Holding Return -- -5.69% if held till today
 Best Month -- January (100% win rate, 25.97% avg gain
 Worst Month -- February (-1.62% avg gain)
 Best Price Band  -- Premium IPOs (>₹1000) gave 18% avg gain
 Best Single IPO -- Highway Infrastructure +72.50%
 Worst Single IPO -- Capital Infra Trust -36.61% 


@ Biggest Insight
 -- Selling IPOs on "listing day gives +9.36% average return"
 -- vs holding till today which gives "-5.69% loss" on average
 --Listing day exit strategy beats long term holding by 15%
